## Installation

To install all dependencies, run:

```bash
pip install -r requirements.txt
```

## Package Descriptions

- **Flask**: Web framework for Python
- **Flask-CORS**: Handle Cross-Origin Resource Sharing
- **Flask-SocketIO**: WebSocket support for Flask
- **python-socketio**: Python client/server implementation of Socket.IO
- **python-engineio**: Python client/server implementation of Engine.IO
- **python-dotenv**: Load environment variables from .env files
- **mysql-connector-python**: MySQL database connector
- **PyJWT**: JSON Web Token implementation
- **Werkzeug**: WSGI utilities library
- **twilio**: Twilio SMS and communications API
- **Flask-Mail**: Email sending support
- **pytest**: Testing framework
- **pytest-mock**: Mocking plugin for pytest
- **pytest-cov**: Coverage plugin for pytest
- **Flask-Limiter**: Rate limiting for Flask
- **pydantic**: Data validation using Python type annotations
- **email-validator**: Email validation library
